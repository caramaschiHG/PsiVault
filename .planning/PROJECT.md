# PsiLock — UI/UX Polish Project

## Vision
Transformar o PsiLock de "funcional mas bagunçado" para "premium, polido e profissional" — um app que psicólogos tenham orgulho de usar diariamente.

## Goals
1. Hierarquia visual clara e consistente em todas as páginas
2. Sistema de componentes reutilizáveis (zero inline styles nas páginas)
3. Responsividade completa (mobile-first)
4. Acessibilidade WCAG 2.1 AA
5. Performance visual (60fps, CLS < 0.05)

## Scope
- App Vault (área logada): inicio, agenda, patients, financeiro, prontuário, settings
- Componentes UI compartilhados
- globals.css (refatoração)
- Landing page (ajustes menores)

## Out of Scope
- Backend/API changes
- Database schema changes
- Auth flow redesign
