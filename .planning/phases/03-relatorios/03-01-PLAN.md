# Phase 03 — Relatórios e Insights

## Objetivo
Dar visão estratégica financeira sem complicar.

## Task Breakdown

### T3.1 — Comparativo mês atual vs anterior
**Descrição:** Nos cards de inadimplência, mostrar variação % vs mês anterior.
**Arquivos afetados:** `src/app/(vault)/financeiro/page.tsx`, `src/app/(vault)/financeiro/page-client.tsx`
**Detalhes:**
- Card "Recebido" mostra: "R$ 3.200 (+12% vs out)" ou "(-5% vs out)"
- Setas ↑↓ com cor verde/vermelho
- Calculado no server component com dados do mês anterior

### T3.2 — Top pacientes por receita
**Descrição:** Seção opcional mostrando quais pacientes mais contribuíram.
**Arquivos afetados:** `src/app/(vault)/financeiro/page-client.tsx`
**Detalhes:**
- Top 5 pacientes por valor recebido no mês
- Lista simples: nome + valor recebido + qtd sessões
- Útil para psicólogo saber quem são seus principais pacientes

### T3.3 — Previsão de receita (sessões agendadas)
**Descrição:** Estimativa de receita baseada em sessões agendadas futuras.
**Arquivos afetados:** `src/app/(vault)/financeiro/page.tsx`, `src/app/(vault)/financeiro/page-client.tsx`
**Detalhes:**
- Conta sessões agendadas (SCHEDULED/CONFIRMED) no resto do mês
- Multiplica pelo preço padrão do paciente
- Mostra: "Previsão: ~R$ 1.800 (12 sessões agendadas)"
- Disclaimer: estimativa, não garantia

### T3.4 — Resumo anual para IR (export)
**Descrição:** CSV com resumo anual para declaração de imposto de renda.
**Arquivos afetados:** `src/app/(vault)/financeiro/actions.ts`, `src/app/(vault)/financeiro/page-client.tsx`
**Detalhes:**
- Botão "Resumo anual" na página financeira
- Exporta CSV com: Mês, Total Recebido, Qtd Sessões, Média por Sessão
- Total anual no rodapé
- Nome do arquivo: `ir-psi-vault-YYYY.csv`
- Formato simples, pronto para contador
