# Plan 25-02: Apresentação do Plano Premium e Assistente (PREM-01, PREM-03)

## Goal
Criar uma página UI apresentando o Assistente de Pesquisa Psicanalítica com expectativas claras, sem features ativas de IA (conceito visual apenas).

## Tasks
1. **Página de Apresentação**:
   - Criar uma nova rota: `src/app/(vault)/vault-plus/page.tsx`.
   - Estruturar a página em formato limpo, editorial, usando os componentes de layout do cofre.
2. **Copy e Framing**:
   - Detalhar "O que faz": literatura, cruzamento de conceitos por pensador, bibliografias comentadas.
   - Detalhar "O que NÃO faz": diagnósticos, análises de casos clínicos, quebra de copyright (apenas metadados para obras protegidas ou links legais).
   - Remover qualquer jargão técnico excessivo de IA ("Prompting", "LLM", etc.).
3. **Navegação**:
   - Inserir um item para essa nova rota na navegação global (`Sidebar` / `BottomNav`) da área logada, usando um ícone discreto (ex: livro, pesquisa).
