# Phase 28-03: Polish — Button :active, Sidebar Transitions, Reduced-Motion

## Goal
Polimento final de transições e feedback visual em todos os pontos de interação.

## Tasks

### Task 1: `.btn-primary:active` consistente
Problema: `.btn-primary` tem `transform` no base mas não tem `:active` com `scale(0.97)` explícito.

Adicionar em globals.css:
```css
.btn-primary:active {
  transform: scale(0.97);
  transition: transform 60ms ease;
}
```

### Task 2: Sidebar collapse com opacidade
Problema: sidebar encolhe mas o conteúdo some sem transição.

Modificar `.vault-sidebar.collapsed`:
```css
.vault-sidebar.collapsed .nav-link {
  opacity: 0;
  transition: opacity 150ms ease;
}
```
Adicionar transição de opacidade nos itens da sidebar.

### Task 3: Sidebar active indicator transition
Verificar que o `box-shadow: inset 2px 0 0` no `.vault-sidebar .nav-link.active` tem transição suave.
Adicionar `box-shadow` à lista de transition properties se necessário.
