## Regra de posicionamento

revalidateTag deve ser chamado:
1. DEPOIS da mutacao no banco (repo.save, repo.update, repo.create, etc.)
2. ANTES do return
3. Junto com o revalidatePath existente (manter ambos)

Exemplo:
```typescript
import { revalidatePath, revalidateTag } from "next/cache";
import { CACHE_TAGS } from "@/lib/cache/tags";

// ... mutation logic ...
await repo.save(updated);
revalidatePath("/financeiro", "page");
revalidateTag(CACHE_TAGS.expenses);
return { ok: true };
```
