## Migration Mapping Table

| Old Value | New Token | Rationale |
|-----------|-----------|-----------|
| 80ms / 0.08s | --duration-100 | Fast micro-interactions |
| 100ms | --duration-100 | Close enough, rounds to canonical |
| 120ms | --duration-100 | Close enough, rounds to canonical |
| 150ms / 0.15s | --duration-200 | Standard interaction |
| 180ms | --duration-200 | Close enough, rounds to canonical |
| 200ms | --duration-200 | Standard interaction |
| 250ms | --duration-300 | Slower, deliberate animations |
| 0.12s | --duration-100 | Equivalent to 120ms |
| ease | --ease-out | Default to ease-out for enters |
| ease-out | --ease-out | Keep easing |
| var(--transition-fast) | var(--duration-100) var(--ease-out) | 80ms ease |
| var(--transition-normal) | var(--duration-200) var(--ease-out) | 150ms ease |
| var(--transition-slow) | var(--duration-300) var(--ease-out) | 250ms ease |

**Exceptions (keep hardcoded):**
- `cubic-bezier(0.16, 1, 0.3, 1)` for .notif-dropdown and .side-panel-drawer — custom easing for specific feel
- `1.4s` for skeleton-shimmer — looping animation outside interaction scale
- `60ms` for button active press — too fast for token scale, keep hardcoded
