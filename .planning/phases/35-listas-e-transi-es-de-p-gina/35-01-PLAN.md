1. Change `.motion-stagger > *` from `fadeIn` to `slideUp` keyframe per D-03:
   ```css
   .motion-stagger > * {
     animation: slideUp var(--duration-200) var(--ease-out) both;
     animation-delay: calc(var(--stagger-index, 0) * var(--stagger-gap));
   }
   ```
   Note: `.motion-stagger` is not yet used in production code, so this change is safe.

2. Add expand/collapse animation classes for `details` elements after the existing utility classes block:
   ```css
   /* Expand/collapse animation for <details> elements */
   .details-animated {
     overflow: hidden;
   }
   .details-animated-content {
     display: grid;
     grid-template-rows: 0fr;
     transition: grid-template-rows var(--duration-300) var(--ease-out);
   }
   .details-animated[open] .details-animated-content {
     grid-template-rows: 1fr;
   }
   .details-animated-content > * {
     overflow: hidden;
     min-height: 0;
   }
   ```

3. Add add/remove animation classes after the expand/collapse block:
   ```css
   /* List item enter/exit animations for client-side lists */
   @starting-style {
     .list-item-start {
       opacity: 0;
       transform: translateY(8px);
     }
   }
   .list-item-start {
     opacity: 1;
     transform: translateY(0);
     transition: opacity var(--duration-200) var(--ease-out), transform var(--duration-200) var(--ease-out);
   }

   .list-item-exit {
     opacity: 0;
     transform: translateY(-8px);
     transition: opacity var(--duration-200) var(--ease-out), transform var(--duration-200) var(--ease-out);
   }
   ```

4. Update the `prefers-reduced-motion: reduce` media query at the bottom of the file to include all new classes. Add to the existing selector list:
   ```css
   .details-animated-content,
   .list-item-start,
   .list-item-exit,
   ```
   Ensure the final rule in the media query block disables transitions too:
   ```css
   @media (prefers-reduced-motion: reduce) {
     /* ... existing selectors ... */
     .details-animated-content,
     .list-item-start,
     .list-item-exit {
       animation: none !important;
       transition: none !important;
       opacity: 1 !important;
       transform: none !important;
     }
   }
   ```
   Note: Add `.details-animated-content,` and `.list-item-start,` to the existing comma-separated selector list (before the closing brace), and add `.list-item-exit` as the last item. The existing block already sets `animation: none !important; opacity: 1 !important; transform: none !important;` — ensure `transition: none !important;` is also present for the new classes.
