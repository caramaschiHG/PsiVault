    - Test 6: `htmlToPdfNodes("<script>alert(1)</script><p>Safe</p>")` ignores script tag and keeps p tag
    - Test 7: Empty string returns empty array

  <action>
    Create `src/lib/documents/html-to-pdf-nodes.ts` that parses sanitized HTML and returns a tree of plain objects representing react-pdf nodes.

    **Parser strategy:** Use a lightweight stack-based parser (no external dependency). The input is already sanitized by `sanitizeRichTextHtml`, so malformed HTML risk is low.

    **Supported tag → node mapping:**
    - `<p>` → `{ type: "view", style: { marginBottom: 6 }, children: [...] }`
    - `<h1>` → `{ type: "view", style: { marginBottom: 10, fontSize: 18, fontFamily: "Helvetica-Bold" }, children: [...] }`
    - `<h2>` → `{ type: "view", style: { marginBottom: 8, fontSize: 14, fontFamily: "Helvetica-Bold" }, children: [...] }`
    - `<strong>` → propagate `bold: true` to child Text nodes
    - `<em>` → propagate `italic: true` to child Text nodes
    - `<u>` → propagate `underline: true` to child Text nodes
    - `<ul>` → `{ type: "view", style: { marginBottom: 6 }, children: [...] }` (each li prefixed with "• ")
    - `<ol>` → same as ul but with numbering "1. ", "2. " etc.
    - `<li>` → `{ type: "view", style: { marginBottom: 2 }, children: [prefix + ...] }`
    - `<br>` → `{ type: "text", text: "\\n" }`
    - `<hr>` → `{ type: "view", style: { borderBottomWidth: 1, borderBottomColor: "#ccc", marginVertical: 8 } }`
    - `<blockquote>` → `{ type: "view", style: { marginLeft: 12, borderLeftWidth: 2, borderLeftColor: "#ccc", paddingLeft: 8 }, children: [...] }`
    - `<a>` → `{ type: "text", text: href, style: { color: "#2563eb", textDecoration: "underline" } }`
    - `<div>` → `{ type: "view", children: [...] }` (transparent container)
    - Text nodes → `{ type: "text", text: decodedText }`

    **Style propagation:** Bold/italic/underline flags accumulate through the parent chain and are applied to leaf Text nodes.

    **Alignment:** Parse `style="text-align:..."` and pass `align: "left" | "center" | "right"` to View style.

    **Export:** `export function htmlToPdfNodes(html: string): PdfNode[]`

    Create `tests/html-to-pdf-nodes.test.ts` with the 7 test cases above. Run `pnpm test` to verify.
  </action>
