  <action>
    Create `src/app/(vault)/patients/[patientId]/documents/components/document-paper-view.tsx` as a Server Component.

    **Props interface:**
    ```typescript
    interface DocumentPaperViewProps {
      typeLabel: string;
      patientName: string;
      professionalName: string;
      crp: string;
      createdAt: Date;
      editedAt: Date | null;
      status: DocumentStatus;
      content: string;
      isPrivate: boolean;
      signatureImageUrl?: string | null;
    }
    ```

    **Layout structure:**
    1. **Outer wrapper:** centered flex container with overflow-x auto for mobile
    2. **Paper element:**
       - `width: 210mm`
       - `minHeight: 297mm`
       - `background: "#fff"`
       - `boxShadow: "0 2px 8px rgba(0,0,0,0.08), 0 0 1px rgba(0,0,0,0.04)"`
       - `padding: "25mm 20mm 30mm"`
       - `fontFamily: "Georgia, 'Times New Roman', serif"`
       - `fontSize: "11pt"`
       - `lineHeight: 1.65`
       - `color: "var(--color-kbd-text)"`
    3. **Header block:**
       - Kicker: "Documento clínico" — uppercase, letterSpacing 1.2px, fontSize 9px, color `var(--color-brown-mid)`
       - Title: `typeLabel` — fontSize 22px, fontFamily sans-serif, fontWeight 700, marginBottom 12px
       - Meta grid: 3-4 rows with label/value pairs (Paciente, Profissional, CRP, Data de criação, Data de edição if present)
       - Meta label: color `var(--color-text-3)`, fontSize 10px, minWidth 120px
       - Meta value: color `var(--color-kbd-text)`, fontSize 11px
    4. **Content block:**
       - marginTop 24px
        - Detect HTML presence: `const hasHtml = content.includes("<") && content.includes(">");`
        - If `hasHtml`: `dangerouslySetInnerHTML={{ __html: sanitizeRichTextHtml(content) }}`
        - Else: render content as preformatted text with `whiteSpace: "pre-wrap"`, fontFamily inherit, lineHeight 1.75
    5. **Signature block:**
       - Render only if `status === "signed"` AND `signatureImageUrl` exists AND `!isPrivate`
       - marginTop 40px, paddingTop 16px, borderTop 1px solid `rgba(146,64,14,0.1)`
       - Signature image: maxHeight 80px, objectFit contain
       - Line: 1px solid `var(--color-border-med)`, width 200px
       - Caption: professionalName + CRP + formatted date
    6. **Watermarks:**
       - Draft (`status === "draft"`): absolute positioned "RASCUNHO" text, rotated -30deg, fontSize 72px, color `rgba(146,64,14,0.06)`, pointerEvents none, centered
       - Archived (`status === "archived"`): similar "ARQUIVADO" stamp, color `rgba(120,113,108,0.08)`

    **Responsive scaling:**
    - Create a small Client Component `document-paper-scaler.tsx` that measures parent container width via ref and applies `transform: scale(ratio)` where `ratio = containerWidth / (210mm in pixels)`.
    - Use `transform-origin: top center`.
    - On viewports >= 210mm + padding, scale is 1 (no transform).
    - Keep `DocumentPaperView` as Server Component for the paper content; the scaler is a thin wrapper around it.
    - The scaler component accepts children and renders them inside a measured container.

    **Security:**
    - Always sanitize HTML via `sanitizeRichTextHtml` before `dangerouslySetInnerHTML`
    - Never render raw content without sanitization
